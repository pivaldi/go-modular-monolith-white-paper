# Contract Files Quick Reference

Quick reference for the auto-generated files in `contracts/go/application/{domain}/`.

## File Layout

```
contracts/go/application/todo/
├── todo_service_contract_gen.go   ← interface + NoopTodoService (generated)
├── errors_gen.go                  ← error code constants (generated)
└── events_gen.go                  ← topic constants + type aliases (generated)
```

```
contracts/go/application/auth/
├── auth_private_service_contract_gen.go  ← AuthPrivateService interface (generated)
├── auth_public_service_contract_gen.go   ← AuthPublicService interface (generated)
├── connect_client.go                     ← PublicHTTPClient + PrivateHTTPClient
├── errors_gen.go                         ← (generated)
├── events_gen.go                         ← (generated)
└── types.go
```

## Service Interface Skeleton (`*_contract_gen.go`)

Generated from the proto `service` block — do not edit directly:

```go
// Code generated by protoc-gen-go-contracts. DO NOT EDIT.
package todo

import (
    "context"
    "fmt"
    todov1 "github.com/pivaldi/mmw-contracts/go/network/todo/v1"
)

type TodoService interface {
    CreateTodo(ctx context.Context, req *todov1.CreateTodoRequest)  (*todov1.CreateTodoResponse, error)
    GetTodo(ctx context.Context, req *todov1.GetTodoRequest)        (*todov1.GetTodoResponse, error)
    UpdateTodo(ctx context.Context, req *todov1.UpdateTodoRequest)  (*todov1.UpdateTodoResponse, error)
    CompleteTodo(ctx context.Context, req *todov1.CompleteTodoRequest) (*todov1.CompleteTodoResponse, error)
    ReopenTodo(ctx context.Context, req *todov1.ReopenTodoRequest)  (*todov1.ReopenTodoResponse, error)
    DeleteTodo(ctx context.Context, req *todov1.DeleteTodoRequest)  (*todov1.DeleteTodoResponse, error)
    ListTodos(ctx context.Context, req *todov1.ListTodosRequest)    (*todov1.ListTodosResponse, error)
}

var ErrTodoServiceUnavailable = fmt.Errorf("todo service unavailable")

type NoopTodoService struct{}
var _ TodoService = (*NoopTodoService)(nil)

func (NoopTodoService) CreateTodo(_ context.Context, _ *todov1.CreateTodoRequest) (*todov1.CreateTodoResponse, error) {
    return nil, ErrTodoServiceUnavailable
}
// … (one method per interface method)
```

## Events Skeleton (`events_gen.go`)

Generated from proto messages with `(options.v1.topic)` annotations:

```go
// Code generated by protoc-gen-go-contracts. DO NOT EDIT.
package todo

import todov1 "github.com/pivaldi/mmw-contracts/go/network/todo/v1"

const (
    TopicUserTaskCreated   = "todo.userTasks.created.v1"
    TopicUserTaskCompleted = "todo.userTasks.completed.v1"
    TopicUserTaskDeleted   = "todo.userTask.deleted.v1"
    // …
)

var Topics = []string{
    TopicUserTaskCreated,
    TopicUserTaskCompleted,
    TopicUserTaskDeleted,
    // …
}

// Type aliases make event types available under the application package.
type UserTaskCreatedEvent = todov1.UserTaskCreatedEvent
type UserTaskCompletedEvent = todov1.UserTaskCompletedEvent
```

## Rule: The Module Implements Its Own Contract

The module's `application.TodoApplicationService` (or a wrapper) must satisfy the
contract interface. A compile-time assertion in the module root file verifies this:

```go
// modules/todo/todo.go
// (The application service satisfies the internal TodoService interface which
//  matches the generated defTodo.TodoService — verified by the contract test suite.)
var _ application.TodoService = (*application.TodoApplicationService)(nil)
```

The external contract test confirms the mapping:
```go
// modules/todo/test/contract/contract_test.go
// Verify the wired module satisfies the external defTodo.TodoService contract.
```

## Generating Contracts

To add a new domain or update after proto changes:

```bash
cd poc/contracts

# 1. Build the custom plugin (once, or after plugin code changes):
go build -o $(go env GOPATH)/bin/protoc-gen-go-contracts ./cmd/protoc-gen-go-contracts

# 2. Generate:
buf generate --template buf.gen.todo.yaml
buf generate --template buf.gen.auth.yaml
```
