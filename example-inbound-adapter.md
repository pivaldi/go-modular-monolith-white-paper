# Inbound Adapter Examples

Inbound adapters (primary adapters) translate external calls into
application service invocations. In this architecture, the primary
inbound adapter is a Connect RPC handler.

## Connect RPC Handler

The handler implements the interface generated by `buf generate` from
the `.proto` definition. It translates between Connect request/response
types and application DTOs.

```go
// modules/foosvc/internal/adapters/inbound/connect/foo_handler.go
package connectadapter

import (
    "context"

    "connectrpc.com/connect"
    "github.com/example/mmw-foosvc/internal/application"
    "github.com/example/mmw-foosvc/internal/application/dto"
    "github.com/example/mmw-foosvc/internal/domain"
    foov1 "github.com/example/mmw-contracts/gen/go/foo/v1"
    "github.com/example/mmw-contracts/gen/go/foo/v1/foov1connect"
)

// FooHandler implements foov1connect.FooServiceHandler (generated interface).
type FooHandler struct {
    svc *application.FooApplicationService
}

// Compile-time assertion: *FooHandler must implement the generated interface.
var _ foov1connect.FooServiceHandler = (*FooHandler)(nil)

func NewFooHandler(svc *application.FooApplicationService) *FooHandler {
    return &FooHandler{svc: svc}
}

func (h *FooHandler) CreateFoo(
    ctx context.Context,
    req *connect.Request[foov1.CreateFooRequest],
) (*connect.Response[foov1.CreateFooResponse], error) {
    ownerID, err := ownerIDFromContext(ctx) // extracted by auth middleware
    if err != nil {
        return nil, connect.NewError(connect.CodeUnauthenticated, err)
    }

    result, err := h.svc.CreateFoo(ctx, dto.CreateFooCommand{
        Title:    req.Msg.Title,
        Priority: req.Msg.Priority,
        OwnerID:  ownerID,
    })
    if err != nil {
        return nil, domainErrToConnect(err)
    }

    return connect.NewResponse(&foov1.CreateFooResponse{
        Foo: fooToProto(result),
    }), nil
}

func (h *FooHandler) GetFoo(
    ctx context.Context,
    req *connect.Request[foov1.GetFooRequest],
) (*connect.Response[foov1.GetFooResponse], error) {
    ownerID, err := ownerIDFromContext(ctx)
    if err != nil {
        return nil, connect.NewError(connect.CodeUnauthenticated, err)
    }

    result, err := h.svc.GetFoo(ctx, dto.GetFooQuery{
        ID:      req.Msg.Id,
        OwnerID: ownerID,
    })
    if err != nil {
        return nil, domainErrToConnect(err)
    }

    return connect.NewResponse(&foov1.GetFooResponse{
        Foo: fooToProto(result),
    }), nil
}
```

## Domain Error → Connect Status Code Mapping

```go
// modules/foosvc/internal/adapters/inbound/connect/errors.go
package connectadapter

import (
    "errors"
    "connectrpc.com/connect"
    "github.com/example/mmw-foosvc/internal/domain"
)

func domainErrToConnect(err error) error {
    switch {
    case errors.Is(err, domain.ErrNotFound):
        return connect.NewError(connect.CodeNotFound, err)
    case errors.Is(err, domain.ErrAlreadyCompleted):
        return connect.NewError(connect.CodeFailedPrecondition, err)
    case errors.Is(err, domain.ErrTitleEmpty),
         errors.Is(err, domain.ErrTitleTooLong):
        return connect.NewError(connect.CodeInvalidArgument, err)
    default:
        return connect.NewError(connect.CodeInternal, err)
    }
}
```

## Auth Middleware

The auth middleware validates a JWT from the `Authorization` header
and injects the authenticated owner ID into the context. It wraps the
entire Connect mux, so every RPC requires authentication.

```go
// modules/foosvc/internal/adapters/inbound/connect/auth_middleware.go
package connectadapter

import (
    "context"
    "net/http"
    "log/slog"

    defbar "github.com/example/mmw-contracts/definitions/barsvc"
)

type contextKey string
const ownerIDKey contextKey = "ownerID"

// NewAuthMiddleware wraps an http.Handler, validates JWTs via the BarSvc
// contract (e.g. the auth module's InprocClient), and injects the owner ID.
func NewAuthMiddleware(
    authSvc defbar.BarService,
    logger *slog.Logger,
    excludedPaths []string,
    next http.Handler,
) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        for _, p := range excludedPaths {
            if r.URL.Path == p {
                next.ServeHTTP(w, r)
                return
            }
        }

        token := r.Header.Get("Authorization")
        if token == "" {
            http.Error(w, "missing authorization header", http.StatusUnauthorized)
            return
        }

        claims, err := authSvc.ValidateToken(r.Context(), token)
        if err != nil {
            logger.Warn("invalid token", "err", err)
            http.Error(w, "invalid token", http.StatusUnauthorized)
            return
        }

        ctx := context.WithValue(r.Context(), ownerIDKey, claims.UserID)
        next.ServeHTTP(w, r.WithContext(ctx))
    })
}
```

## Registration in the Module Factory

The mux, handler, and middleware are assembled in `modules/foosvc/foosvc.go`:

```go
mux := http.NewServeMux()
path, handler := foov1connect.NewFooServiceHandler(
    connectadapter.NewFooHandler(appSvc),
    connect.WithInterceptors(oglconnect.NewErrorLoggingInterceptor(infra.Logger)),
)
mux.Handle(path, connectadapter.NewAuthMiddleware(
    infra.BarSvc, infra.Logger,
    []string{"/health"},  // excluded from auth
    handler,
))
```
