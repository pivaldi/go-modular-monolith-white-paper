# Contract Definition Pattern

In this architecture, contracts are **generated automatically** from proto service definitions
using the custom `protoc-gen-go-contracts` plugin. You write the proto once; the plugin
produces the Go interface, a no-op stub, event topic constants, and error code constants.

The generated contracts live in `contracts/go/application/{domain}/` and are the only
thing other modules import — never a module's `internal/` packages.

## Generated Contract Files

Every domain in `contracts/go/application/{domain}/` contains:

```
contracts/go/application/todo/
├── todo_service_contract_gen.go   ← TodoService interface + NoopTodoService
├── errors_gen.go                  ← error code constants (aliases over proto enum)
└── events_gen.go                  ← TopicXxx constants, Topics slice, event type aliases
```

```
contracts/go/application/auth/
├── auth_private_service_contract_gen.go  ← AuthPrivateService interface + noop
├── auth_public_service_contract_gen.go   ← AuthPublicService interface + noop
├── connect_client.go                     ← PublicHTTPClient + PrivateHTTPClient
├── errors_gen.go
├── events_gen.go
└── types.go
```

## What Each File Contains

### Service Interface (`*_contract_gen.go`)

Generated by `protoc-gen-go-contracts` from the proto `service` block:

```go
// contracts/go/application/auth/auth_private_service_contract_gen.go
// Code generated by protoc-gen-go-contracts. DO NOT EDIT.
package auth

type AuthPrivateService interface {
    ValidateToken(ctx context.Context, req *authv1.ValidateTokenRequest) (*authv1.ValidateTokenResponse, error)
}

var ErrAuthPrivateServiceUnavailable = fmt.Errorf("auth private service unavailable")

type NoopAuthPrivateService struct{}
var _ AuthPrivateService = (*NoopAuthPrivateService)(nil)
func (NoopAuthPrivateService) ValidateToken(_ context.Context, _ *authv1.ValidateTokenRequest) (*authv1.ValidateTokenResponse, error) {
    return nil, ErrAuthPrivateServiceUnavailable
}
```

### Event Topics (`events_gen.go`)

Generated from proto messages that carry `(options.v1.topic)`:

```go
// contracts/go/application/auth/events_gen.go
// Code generated by protoc-gen-go-contracts. DO NOT EDIT.
package auth

const (
    TopicUserRegistered  = "auth.user.registered.v1"
    TopicUserDeleted     = "auth.user.deleted.v1"
    TopicPasswordChanged = "auth.user.password_changed.v1"
    TopicUserLoggedIn    = "auth.user.logged_in.v1"
)

var Topics = []string{TopicUserRegistered, TopicUserDeleted, TopicPasswordChanged, TopicUserLoggedIn}

// Type aliases re-export proto types under the application package.
type UserRegisteredEvent = authv1.UserRegisteredEvent
type UserDeletedEvent    = authv1.UserDeletedEvent
```

### Error Codes (`errors_gen.go`)

Generated from the proto `*ErrorCode` enum:

```go
// contracts/go/application/todo/errors_gen.go
// Code generated by protoc-gen-go-contracts. DO NOT EDIT.
package todo

const (
    ErrorCodeInvalidTitle            = todov1.TodoErrorCode_TODO_ERROR_CODE_INVALID_TITLE
    ErrorCodeNotFound                = todov1.TodoErrorCode_TODO_ERROR_CODE_NOT_FOUND
    ErrorCodeCannotCompleteCancelled = todov1.TodoErrorCode_TODO_ERROR_CODE_CANNOT_COMPLETE_CANCELLED
    // ...
)
```

### HTTP Clients (`connect_client.go`)

Wraps the generated Connect transport — written once, not regenerated:

```go
// contracts/go/application/auth/connect_client.go
package auth

// PrivateHTTPClient adapts the Connect transport to AuthPrivateService.
// Use when auth runs in a separate process.
type PrivateHTTPClient struct {
    client authv1connect.AuthPrivateServiceClient
}

var _ AuthPrivateService = (*PrivateHTTPClient)(nil)

func NewPrivateHTTPClient(client authv1connect.AuthPrivateServiceClient) *PrivateHTTPClient {
    return &PrivateHTTPClient{client: client}
}

func (c *PrivateHTTPClient) ValidateToken(
    ctx context.Context, req *authv1.ValidateTokenRequest) (*authv1.ValidateTokenResponse, error) {
    resp, err := c.client.ValidateToken(ctx, connect.NewRequest(req))
    // ...
}
```

## Dependency Graph

```
contracts/go/application/auth/    ← depends only on mmw-contracts/go/network/auth/v1
        ↑
modules/todo/                     ← imports defauth.AuthPrivateService, calls via in-process accessor
        ↑
cmd/mmw/main.go                   ← creates authModule, calls authModule.PrivateService(),
                                     injects result into todoModule.Infrastructure.AuthSvc
```

`modules/todo` never imports `modules/auth` — only the contract package.
This is enforced by Go's module system: `mmw-todo/go.mod` lists
`mmw-contracts` as a dependency, not `mmw-auth`.

## What Prevents Cycles

The contracts module only depends on `connectrpc/connect` and `google.golang.org/protobuf`.
A feature module can depend on the contracts, but the contracts can never depend on a
feature module. This makes cycles structurally impossible.

```
❌ IMPOSSIBLE (compiler error):
contracts/go/application/auth → modules/auth (would be a circular module dependency)

✅ VALID:
modules/todo → contracts/go/application/auth → (connect + protobuf only)
```

## How todo Uses the Auth Contract

In `modules/todo/go.mod`:
```go
module github.com/pivaldi/mmw-todo
require github.com/pivaldi/mmw-contracts v0.0.0
```

In `modules/todo/internal/adapters/inbound/connect/`:
```go
import defauth "github.com/pivaldi/mmw-contracts/go/application/auth"

func NewTokenValidator(svc defauth.AuthPrivateService) pfmiddleware.TokenValidator {
    return func(ctx context.Context, token string) (uuid.UUID, error) {
        resp, err := svc.ValidateToken(ctx, &authv1.ValidateTokenRequest{Token: token})
        // ...
    }
}
```

In `cmd/mmw/main.go`:
```go
authModule, _ := auth.New(auth.Infrastructure{...})

todoModule, _ := todo.New(todo.Infrastructure{
    AuthSvc: authModule.PrivateService(), // returns defauth.AuthPrivateService (interface)
})
```

## Generating Contracts

```bash
cd poc/contracts

# Build the custom plugin (once):
go build -o $(go env GOPATH)/bin/protoc-gen-go-contracts ./cmd/protoc-gen-go-contracts

# Generate auth contracts:
buf generate --template buf.gen.auth.yaml

# Generate todo contracts:
buf generate --template buf.gen.todo.yaml
```
